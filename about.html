About StadiumAI

Your AI-Powered Stadium Information Assistant

Project Overview

StadiumAI is an AI-powered web application that helps users explore famous sports stadiums. Users can search stadiums, view detailed information, and interact with an AI chatbot for quick answers.

Our Mission

To provide a smarter, safe and more enjoyable stadium experience using Artificial Intelligence and modern web technologies.

Project Features

  • 🤖 AI Chatbot
  • 🗺️ Smart Stadium Navigation
  • 🔐 Admin Login
  • 📅 Match Schedule
  • 🍔 Food Finder
  • 🚨 Emergency Support
  • 📱 Responsive Design

Technologies Used

  • HTML5
  • CSS3 & Bootstrap
  • JavaScript
  • PHP
  • MySQL

Developer

Developed by Ishika Singh for the PromptWars Hackathon as a Smart Stadium Assistant project to provide an interactive platform for exploring stadium information using AI and modern web technologies.
